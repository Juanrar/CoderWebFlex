Python Assembler Simulator

This project is an assembler simulator written in Python that includes implementations of various mathematical functions in assembly language, such as multiplication, division, square root calculation, and the Fibonacci sequence.

Descripcion

The assembler simulator allows the execution of basic assembler instructions using a simple Python virtual machine. The project includes the following features:

  • Basic instructions:

    MOV, ADD, JMP, JNZ, CMP, INC, DEC, NOOP, PUSH, POP, CALL, RET, INT, NEG

  • Mathematical functions:
    Multiplication:
    Implemented using successive sums.
    Division:
    Implemented using successive subtractions.
    Square root:
    Implemented using a trial-and-error approach.
    Fibonacci:
    Implemented using a recursive approach.

Starting with an assembler code file, it is assembled and an executable is created. The function of a processor, operating system, system calls, and includes is emulated.
